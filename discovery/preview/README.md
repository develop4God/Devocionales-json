# Discovery Preview - Spanish Studies

This folder contains markdown (`.md`) previews of the Spanish Bible studies for easy content review before using them in the app.

## Purpose

The markdown files allow you to:
- **Preview content** in a readable format without opening JSON files
- **Review studies** before publishing to the app
- **Share content** with team members for feedback
- **Check formatting** and ensure quality
